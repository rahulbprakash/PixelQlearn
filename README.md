# Pixel-DeepQNetwork

##Overview
 Deep Q Learning network playing/learning single-player pong.

##Dependencies
1. [tensorflow](https://www.tensorflow.org)
2. [cv2](http://www.pyimagesearch.com/2015/06/15/install-opencv-3-0-and-python-2-7-on-osx/)
3. numpy
4. random
5. collections
6. pygame
use [pip](https://pypi.python.org/pypi/pip) to install the dependencies.

##Usage 
Run QLearning.py to begin the learning process.
